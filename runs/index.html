Strava connection

This page calls a serverless proxy that keeps your Strava credentials private. Configure the STRAVA_CLIENT_ID, STRAVA_CLIENT_SECRET, and STRAVA_REFRESH_TOKEN environment variables on your hosting provider (Netlify by default). When hosting the site on GitHub Pages, set PROXY_ENDPOINT in runs/runs.js to your Netlify function URL so the browser only talks to the proxy endpoint and your keys stay off the public site.
